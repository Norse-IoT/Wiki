Basic Internal-Blink example

The following code will toggle the internal led on an ESP32 on or off every second.

From GitHub:

int pin = 2;

void setup() {
  pinMode(pin, OUTPUT);
}

bool state = HIGH;

void loop() {
  digitalWrite(pin, state);
  state = !state;
  delay(1000);
}
